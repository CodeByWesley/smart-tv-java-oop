# Smart TV Simulator (Java)

This is a simple Java project created to practice Object-Oriented Programming (OOP) concepts.

The project simulates the basic behavior of a Smart TV, allowing the user to control the TV power, volume, and channels.

## Features

* Turn the TV on and off
* Increase and decrease volume
* Change channels
* Display the current TV status

## Concepts Practiced

This project was developed to reinforce the following Java fundamentals:

* Classes and Objects
* Methods
* Attributes
* Basic Object-Oriented Programming structure

## Technologies

* Java
* IntelliJ IDEA

## Project Structure

src
├── SmartTv.java
└── Usuario.java

## Purpose

The goal of this project is to strengthen Java fundamentals and practice Object-Oriented Programming while building a portfolio for backend development.
